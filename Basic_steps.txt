
- Create github repo with license (MIT preferred)
- In VSCOde use Ctrl+Shift+P and search Git Clone to clone git repo
    This will create the repo folder where you choose. The folder will contain only README and LICENSE file
- In terminal, navigate to the folder and make a directory named src. 
- Use colcon build in the folder which has src directory. This will create all the necessary file structure
- Add source to bashrc file (source ~/PATH/TO/FOLDER/install/setup.bash)
- Make .gitignore file in main folder to ignore all irrelevant files. (Content to put in git ignore given at end of this document)

-- Create a new package for your project
- In terminal navigate to src folder
- ros2 pkg create MP --build-type ament_python --dependencies rclpy stdmsgs --license MIT 
    Create package named "MP" with ament_python build, dependencies ( very important to mention these) and license

Now the project is ready for user to add nodes and functionality.
